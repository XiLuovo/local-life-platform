# 项目描述（简历版）

## 项目名称

本地生活平台后端

## 项目简介

基于 Spring Boot 独立完成本地生活平台后端整合开发，在外卖点餐系统基础上融合门店浏览、优惠券营销、社交探店和高并发秒杀能力，形成“门店浏览-领券秒杀-下单支付-内容互动”的完整业务闭环。

## 技术栈

- Spring Boot
- MyBatis
- MySQL
- Redis
- JWT
- WebSocket
- Maven

## 简历描述（精简版）

负责将外卖系统与点评秒杀系统整合为本地生活平台后端，完成用户登录鉴权、门店与优惠券、社交笔记与关注、外卖订单和高并发秒杀等核心模块开发；基于 Redis 实现缓存、BitMap 签到、ZSet 点赞与 Feed 流、Lua 原子校验及 Stream 异步下单，提升系统并发处理能力与业务完整度。

## 简历描述（推荐版）

1. 基于苍穹外卖与黑马点评两个项目进行业务整合，设计并实现本地生活平台后端，覆盖用户端、商家后台、营销和社交互动等场景。
2. 重构用户体系，将原有微信登录改为手机号验证码登录 + JWT 鉴权，提升项目本地演示性与面试可讲性。
3. 新增门店、门店分类、优惠券、秒杀券、券订单、探店笔记、关注关系、签到统计等模块，形成“浏览门店-领券秒杀-下单-互动分享”的完整链路。
4. 基于 Redis 实现门店缓存、BitMap 连续签到、Set 关注关系、ZSet 点赞与关注 Feed 推送。
5. 将秒杀下单升级为 Redis Lua + Stream 异步处理方案，实现库存校验、一人一单原子控制、消息异步落库和 pending-list 补偿。
6. 使用 WebSocket 支持订单催单/通知场景，并为无 `openid` 用户补充 mock 支付方案，保证本地环境可完整演示业务流程。
